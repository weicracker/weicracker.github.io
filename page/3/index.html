python常用模块logging、shutil

发表于 2017-08-20 | 分类于 python

常用模块

日志管理 ： 「logging」

  • 简介

logging模块是Python内置的标准模块，主要用于输出运行日志，可以设置输出日志的等级、日志保存路径、日志文件回滚等；相比print，具备如下优点：

  1. 可以通过设置不同的日志等级，在release版本中只输出重要信息，而不必显示大量的调试信息；
  2. print将所有信息都输出到标准输出中，严重影响开发者从标准输出中查看其它数据；logging则可以由开发者决定将信息输出到什么地方，以及怎么输出；

简介

在github发布开源项目时，开始经常会遇到不同程度的小问题。这篇文章将对在使用github中遇到的问题，提供解决方案

问题及解决方案

如何修改github上仓库的项目语言 ?

解决办法

项目根目录添加 「.gitattributes」 文件

作用：把项目中.bat 文件识别为以.py结尾的文件

项目中有些文件或文件夹不想上传至github或不想将这些文件进行版本控制

解决办法

项目根目录添加 「.gitignore」 文件

作用： 忽略掉以「.pyc」结尾的文件及「build」文件夹下的所有文件
