常用文件夹命名

发表于 2015-08-31 
在软件开发过程中，经常遇到文件夹不知道如何命名，并且没有统一的命名规范，总感觉文件夹命名
很乱。这篇文章主要介绍了，本人在开发过程中常用的文件夹命名，及使用介绍。

常用文件夹命名及简介

名称 简介 名称 简介
tests 测试文件 src 源码文件
components 组件 libs 静态文件
docs 介绍文档 css 样式表
js javascript examples 运行示例
images 图片文件 dist 可执行文件
assets 资源文件 utils 工具脚本
bin 可执行脚本 fixtures 特征文件
fonts 字体文件 build 构建脚本
core 主文件 helpers 帮助文件
plugins 插件 config 配置文件
data 测试数据 public 公共文件
themes 主题文件 services 服务文件
html html i18n 国际化
